이 프로젝트는 OpenCV를 이용해 카메라로 촬영한 이미지에서 문서를 자동으로 인식하고, 스캔된 것처럼 보이도록 변환해주는 프로그램입니다.

스마트폰 스캐너 앱(CamScanner 같은 것)의 핵심 기능을 파이썬으로 구현한 것입니다.

실시간 카메라 영상 처리
문서(종이) 자동 탐지
기울어진 문서 보정 (Perspective Transform)
스캔한 것 같은 흑백 이미지 생성
키보드로 스캔 및 종료 제어

1️⃣ preprocess.py

이미지를 문서 탐지에 적합하게 변환하는 단계입니다.

컬러 이미지를 흑백으로 변환
Gaussian Blur로 노이즈 제거
Canny Edge Detection으로 윤곽선 추출

👉 결과: 문서 테두리를 쉽게 찾을 수 있는 edge 이미지 생성

2️⃣ detector.py

이미지에서 문서 형태(사각형)를 찾는 역할입니다.

윤곽선(contour) 검출
가장 큰 윤곽선부터 정렬
4개의 꼭짓점을 가진 형태만 문서로 판단

👉 결과: 문서의 4개 꼭짓점 좌표 반환

3️⃣ utils.py

문서의 좌표를 정리하는 도구입니다.

4개의 점을
좌상 (top-left)
우상 (top-right)
우하 (bottom-right)
좌하 (bottom-left)

순서로 정렬합니다.

👉 원근 변환을 위한 필수 단계

4️⃣ transform.py

문서를 “스캔된 형태”로 만드는 핵심 단계입니다.

✔ warp_perspective
기울어진 문서를 정면으로 변환
실제 스캐너처럼 평평하게 보이도록 보정
✔ get_scanned_effect
흑백 변환
adaptive threshold 적용
종이 느낌의 스캔 이미지 생성

👉 결과: 실제 스캔본 같은 이미지 생성

5️⃣ main.py (실행 파일)

전체 프로그램을 제어하는 메인 루프입니다.

기능
카메라 실행
실시간 화면 출력
문서 탐지 결과 표시 (초록색 박스)
SPACE: 문서 스캔 실행
Q: 프로그램 종료
